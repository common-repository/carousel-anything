== Description ==
This plugin is a extension for WPBakery Page Builder (Formerly Visual Composer) which will allow user to display any content in slider.You can use it as a <b>Image slider,Post Slider,Testimonial Slider,Product Slider</b> and many more. It is very easy to use and no knowledge of coding required. It is 100% responsive and touch enabled on touch devices. It comes up with beautiful animation styles which add interactivity to the site and engage user in it.

<a href="https://demo.webdevocean.com/carousel-anything-for-wpbakery-page-builder/">See DEMO Here</a><br>
